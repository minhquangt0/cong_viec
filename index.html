業務委託契約 変更合意書
（AIオーダーシステム開発に関する契約内容の変更について）

tiffin株式会社（以下「甲」という。）とAMCOLAB TECHNOLOGY Co., Ltd（以下「乙」という。）は、2025年8月15日付で締結した「システム開発業務委託契約書」（以下「原契約」という。）について、以下のとおり変更することに合意する。

第1条（契約変更の目的）

本変更合意書は、原契約に定める開発業務の範囲、フェーズ構成、開発費用、納期及び支払条件等を再定義し、より実態に即した開発体制を確立することを目的とする。

第2条（開発範囲の変更）

原契約第3条に定める本件開発業務の内容を、以下のとおり変更する。

1. 開発は以下の3つのフェーズで構成される：

【Phase 0：要件定義・UI/UXデザイン】

  • 要件仕様フローの作成
  • 画面モックアップの作成
  • UI/UXデザイン制作

【Phase 1：MVPシステム開発】

Phase 0の成果物に基づき、1店舗での実運用が可能な基本システムを開発する。
主要機能：

  • AI音声・テキスト注文機能
  • Square決済連携
  • キッチン管理システム（注文受信、調理進捗管理、優先順位表示）
  • 基本管理画面（店舗設定、スタッフ管理、顧客管理、メニュー管理）
  • Webアプリ（QR注文、メニュー表示、決済連携）
  • レコメンデーション機能
  • 統計・分析ダッシュボード
  • レストラン統合管理ボード
  • 注文レポート機能

【Phase 2：拡張機能開発】

Phase 1の成果物を基盤とし、以下の拡張機能を開発する：

  • PayPay決済連携
  • 店内順番待ち管理システム
  • ホールスタッフオペレーション機能
  • LINE連携（予約機能）
  • 基本レポート機能（顧客レポート、売上レポート）

2.詳細な機能仕様は、別紙「AIオーダーシステム開発フェーズ提案書」に準拠する。

第3条（除外機能）

以下の機能は、Phase 0、Phase 1、Phase 2のいずれにも含まれず、本変更合意書の対象外とする：

  • 音声認識システム（ASR）の独自開発
  • 音声生成サーバ構築（TTS）
  • 中国語・韓国語対応の自然言語処理
  • 中国語・韓国語のUI対応（Mobile/Web）
  • アレルギー警告機能
  • クーポン・ポイント連携
  • フィードバックシステム
  • ハンズフリー操作（厨房スタッフ向け）
  • ガイド付き注文機能
  • 音声ロイヤリティプログラム
  • BGM/環境音調整機能
  • 時間帯連動メニュー
  • スタッフ間メッセージ機能
  • テンプレ回答機能
  • スタッフトレーニングモード
  • 音声感情解析機能
  • カメラ学習機能
  • 既存モバイルオーダー統合
  • シフト管理機能
  • 精算・決済管理機能
  • テナント管理機能
  • 店舗管理機能（複数店舗向け）
  • 伝票出力機能（ドリンク・フード）

上記除外機能について、将来的に甲が開発を希望する場合は、甲乙協議の上、別途見積を作成し、新たな契約又は変更契約を締結するものとする。

第4条（開発費用の変更）

1. 原契約第16条第1項に定める報酬総額を、以下のとおり変更する：

フェーズ 工数（時間） 費用（円）
Phase 0 + Phase 1（MVP） 3,430時間 ¥6,174,000
Phase 2（拡張） 1,014時間 ¥1,826,000
合計 4,444時間 ¥8,000,000

2. 上記費用は、10％の割引を適用した後の金額である（割引前：¥8,888,000）。

3. 上記費用は非課税である。乙は外国法人であるため、日本国内における税務上の義務は原則として発生しない。万一、本契約に関連して日本国内で何らかの税務上の義務が発生した場合、当該税務上の義務は甲が負担するものとする。

第5条（納期の変更）

原契約第10条に定める納入期限を、以下のとおり変更する。

1. Phase 1（Phase 0の成果物を含む）

開発期間：12月初旬から3月末まで

UAT期間：3月から4月まで

備考：Phase 0（モックアップ・UI/UX）は、Phase 1の開発期間内で並行して実施し、Phase 1の初期段階で完了する。詳細スケジュールは、AMCOLABが提示するプロジェクト計画書に従う。

2. Phase 2

開始時期：3月初旬（甲乙双方がPhase 1の開発結果を確認し、Phase 2開始について合意した後）

開発期間：3月から5月末まで

UAT期間：5月から6月まで

3. スケジュール調整

甲乙双方は、上記の期間が予定であることを理解し、以下の事由により合理的な範囲で調整できることに合意する：

  • 仕様・設計の確認状況
  • 甲側からのUATフィードバックの進捗状況
  • その他の客観的事情

スケジュールに重大な変更が生じる場合は、甲乙双方が電子メール又は書面により協議の上、確認するものとする。

第6条（支払条件の変更）

原契約第16条第2項に定める支払条件を、以下のとおり変更する。

1. Phase 0 + Phase 1（MVP）

Phase 0及びPhase 1を一括して管理し、合計金額は¥6,174,000とする。

  • 支払済：¥2,982,600（甲は既に当該金額を乙に支払済である）
  • 検収完了後の支払：¥3,191,400（Phase 1の検収完了後、甲は乙に対し当該残額を支払う）

2. Phase 2（拡張機能）

Phase 2の合計金額は¥1,826,000とする。

  • 第1回支払（Phase 2着手時）：¥913,000（Phase 2費用の50％）
  • 第2回支払（検収完了後）：¥913,000（Phase 2費用の50％）

3. 検収及び支払期限

各フェーズの検収は、原契約第12条及び第13条の規定に準拠して行うものとする。

第7条（成果物の変更）

原契約別紙「成果物目録」を、本変更合意書別紙「AIオーダーシステ開発Phase提案書」記載の成果物に変更する。具体的には、各フェーズ完了時に以下の成果物を納入するものとする：

  • ソースコード一式
  • 面面モック・Figmaデザイン
  • API仕様書
  • システム構成図、DB設計図、ER図
  • システム引き渡し資料
  • ステージング環境及び本番環境への移行作業

第8条（原契約の適用）

本変更合意書に定めのない事項については、原契約の規定を適用する。本変更合意書と原契約の内容が矛盾する場合は、本変更合意書の内容が優先するものとする。

第9条（効力発生日）

本変更合意書は、甲⼄両者が署名⼜は記名押印（電⼦的な署名⼜は押印を含む）した⽂書をPDF形式にて電 ⼦メールで相互に送受信すること、または、署名若しくは記名押印に代わる電磁的処理を施すことに よって締結するものとする。甲及び⼄は、電磁的記録（電⼦データ）を本変更合意書の原本とすることに同 意し、それを各⾃保存するものとする。
